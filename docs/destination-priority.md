# 여행지 우선순위 운영 기준

## 목적

전 세계를 같은 깊이로 동시에 수집하지 않고 한국인 여행 수요가 큰 국가, 지역, 도시부터 공식 출처와 검수 데이터를 확장한다.

## 근거

- 한국관광 데이터랩의 국민 해외관광객 및 주요 목적지 공개 통계
- 서비스가 보유한 공식 출처 수, 검수 상태, 최근 확인일
- 항공 접근성과 한국인 대상 여행 수요가 높은 지역에 대한 편집 판단

`priorityScore`는 공식 방문객 순위가 아니다. 공개 통계의 범위와 집계 방식에 한계가 있으므로 서비스 운영을 위한 편집 우선순위로만 사용한다.

## 점수와 상태

- 90 이상: 최우선 수집·검수 대상
- 70~89: 주요 확장 대상
- 1~69: 순차 확장 대상
- 0: 근거 검토 전

콘텐츠 상태는 `AVAILABLE`, `PARTIAL`, `IN_REVIEW`, `REQUESTED`, `NO_VERIFIED_DATA`로 관리한다. 점수는 정렬과 인기 표시에만 사용하며 법적 중요도나 안전 위험도를 의미하지 않는다.

## 갱신

분기마다 공개 통계의 기준 시점, 공식 출처 보유량, 검수 완료 수를 확인하고 `prisma/seed-data/travel-priorities.mjs`를 갱신한다. 변경 시 근거와 기준일을 PR 본문에 기록한다.

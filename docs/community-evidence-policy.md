# 여행객 후기·커뮤니티 기반 문화 신호 정책

## 목적

공식 관광 안내만으로 포착하기 어려운 암묵적 규칙, 상황별 민감도, 실제 부작용을 여행자 후기와 현지인 토론에서 발견한다.

## 허용 출처

- Reddit 등 공개 커뮤니티 토론
- 여행 전문 매체와 현지 생활 매체
- 여행 후기 블로그
- 나무위키 등 사용자 편집 위키

이 출처들은 문화 신호를 찾는 보조 자료이며 법률·벌금·형량의 근거로 사용하지 않는다.

## 신뢰 등급

- `OFFICIAL`: 정부, 관광청, 세관, 교통기관 등 공식 출처
- `CORROBORATED`: 공식·전문 매체와 서로 독립된 여행자 경험이 같은 맥락을 지지
- `COMMUNITY_SIGNAL`: 여러 경험에서 반복되지만 공식 근거가 없거나 상황 차이가 큰 신호

`COMMUNITY_SIGNAL`은 자동으로 `VERIFIED` 처리하지 않고 `REVIEWING` 상태로만 저장한다.

## 작성 기준

1. 최소 2개 독립 출처에서 비슷한 경험이 반복돼야 한다.
2. “왜 문제인지”, “실제로 어떤 반응이 생기는지”, “언제 덜 민감한지”를 함께 기록한다.
3. 지역·세대·시설에 따른 반례를 `counterpoint`에 기록한다.
4. 국가 전체의 성격으로 일반화하지 않는다.
5. 커뮤니티 댓글을 그대로 복제하지 않고 요약한다.
6. 특정 개인, 민족, 종교를 비하하거나 고정관념을 강화하는 항목은 제외한다.

## UI 표시

- 공식 출처와 여행자 후기 출처를 서로 다른 라벨로 표시한다.
- `COMMUNITY_SIGNAL`은 “여행자 경험 기반 · 검수 중”으로 표시한다.
- 독립 출처 수와 과장 방지 문구를 상세 화면에서 확인할 수 있게 한다.
